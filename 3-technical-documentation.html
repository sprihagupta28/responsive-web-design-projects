Introduction

JavaScript is a high-level, interpreted programming language that runs in every modern web browser. It powers interactivity on the web, from form validation to full single-page applications.

This guide covers the core building blocks a beginner needs to start writing JavaScript. You can run every example below directly in your browser's developer console.

JavaScript is case-sensitive and uses the Unicode character set. Statements are typically terminated with a semicolon, though it is inserted automatically in most cases.

Variables

Variables are named containers for storing data values. In modern JavaScript there are three ways to declare them:

  • let — a block-scoped variable that can be reassigned.
  • const — a block-scoped constant that cannot be reassigned.
  • var — the older, function-scoped declaration.

A simple declaration looks like this:

let score = 10; const name = "Ada"; score = 20; // allowed // name = "Grace"; // TypeError: cannot reassign a const

As a rule of thumb, prefer const by default and use let only when you know a value will change.

Data Types

JavaScript has several primitive data types plus objects. The primitives are the simplest values the language works with.

  • String — text, e.g. "hello"
  • Number — integers and decimals, e.g. 42 or 3.14
  • Boolean — true or false
  • Null, Undefined — absence of a value

You can check any value's type with the typeof operator:

typeof "hello"; // "string" typeof 42; // "number" typeof true; // "boolean" typeof {}; // "object"
Functions

Functions are reusable blocks of code that perform a task and can return a value. They are the fundamental unit of composition in JavaScript.

function add(a, b) { return a + b; } // Arrow function equivalent const multiply = (a, b) => a * b; add(2, 3); // 5 multiply(2, 3); // 6

Functions can take parameters, provide default values, and be passed around as first-class values — meaning you can store them in variables and pass them to other functions.

Conditionals

Conditional statements let your program make decisions and run different code depending on whether a condition is true or false.

const hour = 14; if (hour < 12) { console.log("Good morning"); } else if (hour < 18) { console.log("Good afternoon"); } else { console.log("Good evening"); }

For short choices you can use the ternary operator: const label = age >= 18 ? "adult" : "minor";

Loops

Loops run a block of code repeatedly until a condition is met. They are essential for working with collections of data.

// Classic for loop for (let i = 0; i < 3; i++) { console.log(i); // 0, 1, 2 } // Iterating an array const fruits = ["apple", "pear", "kiwi"]; for (const fruit of fruits) { console.log(fruit); }

Other common loop forms include while, do...while, and array methods like forEach and map.
